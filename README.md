# Cadastro e Listagem de itens

Requisitos
- node v8 
- npm v 5.6.0

Execução
1 - Execute npm install para instalar os pacotes da aplicação

2 - Execute npm run start para executar o app em modo desenvolvimento. Abra o navegador em http://localhost:4200/.
